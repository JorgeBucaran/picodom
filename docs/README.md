## Installation

With npm/Yarn.

<pre>
npm i [picodom](https://npmjs.com/package/picodom)
</pre>

From a CDN.

```html
<script src="https://unpkg.com/picodom"></script>
```

## Usage

With ES5/ES6 via a bundler.

```jsx
import { h, patch } from "picodom"
```

With a <samp>\<script\></samp> tag.

```js
const { h, patch } = picodom
```
